## Application List

### Brew

- `kubernetes-cli`: Kubernetes command-line interface, <https://kubernetes.io/>
- `helm`: Kubernetes package manager, <https://helm.sh/>
- `kind`: kind is a tool for running local Kubernetes clusters using Do`cker container “nodes”, <https://kind.sigs.k8s.io/>
- `kustomize`: Customization of kubernetes YAML co`nfigurations, required for kubebuilder, <https://github.com/kubernetes-sigs/kustomize>
- `istioctl`: Istio configuration command-line utility, <https://istio.io/latest/docs/reference/commands/istioctl>
- `argocd`: ArgoCD, <https://argoproj.github.io/cd>
- `kubeseal`: Kubernetes controller and tool for one-way encrypted Secrets, <https://github.com/bitnami-labs/sealed-secrets>
- `dive`: Tool for exploring each layer in a docker image, <https://github.com/wagoodman/dive>
- `k9s`: Kubernetes CLI To Manage Your Clusters In Style!, <https://k9scli.io/>
